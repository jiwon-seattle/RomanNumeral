## Description

**An application that takes string symbols and returns integer values.**

## Specifications

Roman numerals are based on seven symbols:
| Characters | Scrabble Values | 
| :-------------     | :------------- | 
| I | 1 |
| V | 5 |
| X | 10 |
| L | 50 |
| C | 100 |
| D | 500 |
| M | 1,000 |
